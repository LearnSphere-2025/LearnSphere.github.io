HR-Chatbot

Conversational Assistant for Labor Risk & HR Policy Insights

Chatbot RAG FastAPI Legal AI

Overview

HR-Chatbot is an interactive web-based assistant that provides real-time HR and labor law insights to professionals, small business owners, and researchers.

It builds upon the HR-RAG system to deliver contextual answers, policy summaries, and compliance guidance through natural dialogue.

Core Features

  • Conversational Q&A on HR compliance, employee relations, and workplace policy
  • Source-linked responses — every answer cites retrieved documents for verification
  • Continuous learning loop — integrates user feedback into retrieval ranking
  • Web Interface built with FastAPI + Streamlit, connected to a scalable LLM backend

Use Cases

  • HR consultants drafting compliance checklists
  • Labor attorneys reviewing recent case precedents
  • Startups managing HR risk without in-house legal staff

Technical Highlights

Component Stack
Backend FastAPI + LangChain + Milvus (vector store)
Frontend Streamlit + Tailwind UI
Model Fine-tuned LLM (KoRAG-base)
Deployment AWS Lambda + GitHub Actions

Future Roadmap

  • Adaptive Dialogue Memory with topic persistence
  • Multi-modal input (e.g., PDF upload for policy analysis)
  • Integration with HR-RAG API for report generation on request
"HR-Chatbot brings domain-specific intelligence to workplace conversations — bridging law, policy, and AI."
