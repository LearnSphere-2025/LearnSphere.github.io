HR Consultant Web

Corporate HR Risk Self-Audit & AI-Powered Custom Analysis System

Self-Audit AI Analysis Consulting Risk Management

Overview

HR Consultant Web is a web service that allows companies to self-audit their HR risks and provides AI-powered custom analysis reports in real-time.

By simplifying complex labor audits, users can quickly identify key risks and check response strategies without expert assistance.

Key Features

  • Simple/Detailed Self-Audit — Systematic risk check via 13 (Simple) or 189 (Detailed) questions
  • AI Smart Analysis — Mapping to 16 key areas and automatic diagnosis of 5 vulnerability types (Persona)
  • Real-time Solutions — Immediate improvement directions and Action Plans for identified issues
  • Expert Consultation Connection — One-stop labor attorney consultation request via Google Forms/Sheet integration

Use Cases

  • Early-stage Startups: When basic labor risk checks are needed without a dedicated HR manager
  • SMEs/Mid-sized Companies: Checking for omissions in mandatory legal obligations such as employment rules and labor contracts
  • HR Managers: Quickly diagnosing the organization's overall HR health and creating reports

Tech Stack

Component Stack
Frontend Vanilla JavaScript + HTML5 + CSS3 (Vite)
Backend/DB Google Sheets API + Apps Script (Serverless)
AI Logic Client-side Scoring Engine + Rule-based Analysis
Deployment Vercel / Netlify (Static Hosting)

Future Roadmap

  • AI Chatbot Introduction: Conversational interface to ask and receive answers about analysis results in natural language
  • PDF Report Download: Converting diagnostic results into formal reports and providing downloads
